### 2. 类型定义

**完成时间**: 2026-04-29

**文件**: `src/types/api.ts`
- `ApiResponse<T>`: 统一 API 响应格式
- `PaginatedResponse<T>`: 分页响应格式

**文件**: `src/types/user.ts`
- `User`: 用户信息类型
- `LoginRequest`: 登录请求
- `RegisterRequest`: 注册请求
- `AuthResponse`: 认证响应

**文件**: `src/types/interview.ts`
- `Position`: 面试岗位枚举（golang, java, frontend, test）
- `Direction`: 面试方向枚举（backend, cloud, agent, server）
- `InterviewStage`: 面试阶段枚举（intro, questioning, algorithm, closing, finished）
- `ProgrammingLanguage`: 编程语言枚举（java, python, go, cpp）
- `InterviewConfigRequest/Response`: 面试配置
- `CreateInterviewRequest/Response`: 创建面试
- `InterviewStateResponse`: 面试状态查询
- `AudioSubmitHeaders/Response`: 音频提交
- `CodeSubmitRequest/Response`: 代码提交
- `FinishInterviewRequest/Response`: 结束面试
- `InterviewTurn`: 面试对话轮次
- `SSEEventType`: SSE 事件类型枚举
- `TextDeltaEvent/TextDoneEvent`: 文字流事件
- `AudioDeltaEvent/AudioDoneEvent`: 音频流事件
- `StageChangedEvent`: 阶段切换事件
- `CodeJudgedEvent`: 代码评估事件
- `ResumeParsedEvent`: 简历解析事件
- `ReportReadyEvent`: 报告生成事件
- `InterviewFinishedEvent`: 面试结束事件
- `SSEErrorEvent`: 错误事件

**文件**: `src/types/resume.ts`
- `ResumeEducation`: 教育背景（学校、专业、毕业时间）
- `ResumeProject`: 项目经验（项目名、技术栈、描述、亮点）
- `ResumeInternship`: 实习经历（公司、职位、描述）
- `StructuredResume`: 结构化简历（技能、项目、实习、教育）
- `ResumeSubmitRequest`: 简历提交请求（表单方式）
- `ResumePDFUploadRequest`: PDF 上传请求
- `ResumeParseResponse`: 简历解析响应

**文件**: `src/types/report.ts`
- `ReportStatus`: 报告状态枚举（pending, generating, done, failed）
- `ReportDimensions`: 报告维度评分（知识深度、表达能力、问题解决、代码质量、压力应对）
- `ReportStatusResponse`: 报告状态查询响应
- `ReportResponse`: 报告详情（维度评分、总结、优势点、待改进点）
- `ReportQueryParams`: 报告查询参数

**数据流转**: 
```
API 响应 → ApiResponse<T> → 业务数据 T
用户表单/PDF → ResumeSubmitRequest/ResumePDFUploadRequest → API → StructuredResume → resumeStore
面试结束 → 报告生成 → ReportResponse → 雷达图展示
```

## 待完成模块

### 3. 状态管理

**完成时间**: 2026-04-29

**文件**: `src/store/authStore.ts`
- `useAuthStore`: 认证状态管理
- 状态：user, token, isAuthenticated, isGuest
- 方法：setUser, setToken, setGuest, logout, clearAuth
- 持久化：localStorage（auth-storage）

**文件**: `src/store/interviewStore.ts`
- `useInterviewStore`: 面试状态管理
- 状态：interviewId, stage, position, direction, programmingLanguage, turns, isConnected, isProcessing
- 方法：setInterviewId, setStage, setPosition, setDirection, setProgrammingLanguage, addTurn, updateLastTurn, setConnected, setProcessing, reset

**文件**: `src/store/deviceStore.ts`
- `useDeviceStore`: 设备检测状态管理
- 状态：microphonePermission, cameraPermission, microphoneDeviceId, cameraDeviceId, microphoneStream, cameraStream, isMicrophoneTested, isCameraTested, audioLevel
- 方法：setMicrophonePermission, setCameraPermission, setMicrophoneDeviceId, setCameraDeviceId, setMicrophoneStream, setCameraStream, setMicrophoneTested, setCameraTested, setAudioLevel, stopAllStreams, reset

**文件**: `src/store/resumeStore.ts`
- `useResumeStore`: 简历状态管理
- 状态：resume, isParsing, parseError
- 方法：setResume, updateResume, setParsing, setParseError, clearResume
- 持久化：localStorage（resume-storage）

**数据流转**:
```
用户操作 → store actions → 状态更新 → UI 重渲染
authStore/resumeStore → localStorage 持久化
```

### 6. 通用组件库

**完成时间**: 2026-04-29

**基础组件** (`src/components/common/`):

- `Button.tsx`: 按钮组件
  - Props: variant (primary/secondary/text), size (sm/md/lg), loading
  - 支持加载状态、禁用状态
  
- `Input.tsx`: 输入框组件
  - Props: label, error, helperText, type
  - 密码输入支持显示/隐藏切换
  
- `Select.tsx`: 下拉选择组件
  - Props: label, error, options, placeholder
  - 支持选项列表配置
  
- `FileUpload.tsx`: 文件上传组件
  - Props: accept, maxSize, onFileSelect
  - 支持拖拽上传、文件大小限制
  
- `Modal.tsx`: 弹窗组件
  - Props: isOpen, onClose, title, footer, size
  - 支持标题、底部操作区、尺寸配置
  - 自动管理 body overflow
  
- `Loading.tsx`: 加载动画组件
  - Props: size, text, fullScreen
  - 支持全屏加载和局部加载
  
- `Toast.tsx`: 消息提示组件
  - Props: message, type (success/error/warning/info), duration
  - 自动消失、支持手动关闭
  
- `Card.tsx`: 卡片容器组件
  - Props: title, padding (none/sm/md/lg)
  - 支持标题和内边距配置

**布局组件** (`src/components/layout/`):

- `Header.tsx`: 顶部导航
  - 显示系统标题
  - 显示用户信息（用户名/游客模式）
  - 退出登录按钮
  
- `Footer.tsx`: 底部信息
  - 版权信息
  
- `Container.tsx`: 页面容器
  - Props: showHeader, showFooter, maxWidth
  - 统一页面布局（Header + Main + Footer）
  - 支持最大宽度配置

**样式规范**:
- 使用 Tailwind CSS
- 主题色：primary-500 (#3b82f6)
- 圆角：rounded-lg
- 阴影：shadow-md / shadow-lg
- 过渡动画：transition-colors duration-200
